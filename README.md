# Anshul Kamble Portfolio

## Build Context

This project is built extensively with `Codex GPT-5.4` throughout the design, frontend implementation, iteration, and refinement process.

Single-page portfolio website built as a static frontend for `anshulkamble.com`.

## Tech Stack

- `HTML5`
- `CSS3`
- `Vanilla JavaScript`
- `Font Awesome` for icons
- `Google Fonts` (`DM Sans`, `JetBrains Mono`)

## Frontend Notes

- Single-page architecture with smooth-scroll section navigation
- Responsive layout using CSS Grid, Flexbox, and media queries
- Sticky navigation with active-section highlighting
- Reusable card-based design system across hero, experience, projects, skills, education, and campus life
- Lightweight interactive UI patterns including hover elevation, modal image previews, and timeline/card layouts

## Libraries / External Resources

- `Font Awesome 6.5.2`
- `Google Fonts CDN`
